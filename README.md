## Installation

### For linux

> Prerequisite:
> stow installed on your path

```bash
git clone https://github.com/sylvan-lyon/dotfiles.git ~/.dotfiles \
    cd ~/.dotfiles
    ./deploy/linux.sh
```

### For Windows

```powershell
git clone https://github.com/sylvan-lyon/dotfiles.git ~/.dotfiles \
    cd ~/.dotfiles
    .\deploy\windows.ps1
```
